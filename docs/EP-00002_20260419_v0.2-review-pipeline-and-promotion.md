# EP-00002 — v0.2 Review Pipeline & Promotion

## Problem / Pain Points
- v0.1 stores sessions and turns but they never become knowledge — the pipeline stops at raw data
- No review tools exist to synthesize session drafts into knowledge drafts
- No promotion tools to move drafts into curated knowledge or skills
- `session_start` context loading is stubbed — agents don't receive relevant skills/knowledge on startup
- No CLI inspection or review commands for users

## Suggested Solution
Complete the knowledge maturation pipeline: session review → daily review → promotion → context loading. Add CLI commands for review and inspection. This makes the system end-to-end functional.

## Implementation Phases

### Phase 1 — Review Tools
- [ ] `review_get_pending` MCP tool — returns (1) orphaned sessions with raw turns (no session draft), (2) unreviewed session drafts from previous days
- [ ] `review_complete` MCP tool — sets `reviewed_at` on sessions, deletes processed session draft files
- [ ] Wire orphan auto-close in `session_start` (storage already has `close_orphaned_sessions`, ensure it runs and pending status is accurate)
- [ ] Tests for review tools

### Phase 2 — Promotion Tools
- [ ] `promote_to_knowledge` MCP tool — move file from `/memory/drafts/knowledge/` to `/memory/knowledge/`, record memory_edit, update search index
- [ ] `promote_to_skill` MCP tool — move file from `/memory/knowledge/` to `/memory/skills/`, record memory_edit, update search index
- [ ] Tests for promotion tools

### Phase 3 — Session Context Loading
- [ ] Improve `session_start` recommended context — return actual content of matching skill pages based on project tags
- [ ] Return recent knowledge pages related to the project (search by project name/tags)
- [ ] Tests for context loading

### Phase 4 — CLI Review & Inspection
- [ ] `akw review` — fetch pending session drafts, call configured LLM to synthesize knowledge drafts, write to `/memory/drafts/knowledge/` and review report to `/memory/drafts/reviews/`
- [ ] `akw sessions [--project X] [--date Y]` — list sessions with summaries
- [ ] `akw search "query" [--tier T]` — search memory from terminal
- [ ] Tests for CLI commands

## Out of Scope (EP-00003+)
- MCP prompts (`session-bootstrap`, `session-wrapup`)
- Secret scanning (write-time content sanitization)
- `maintain_get_stats`, `maintain_purge`
- CLI `akw purge`, `akw reindex`
