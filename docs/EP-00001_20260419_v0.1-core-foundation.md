# EP-00001 — v0.1 Core Foundation

## Problem / Pain Points
- No project structure exists yet — need scaffolding before any feature work
- Core library (storage, file ops, search) must be in place before MCP server or CLI can be built
- Need a working MCP server with basic tools to validate the architecture end-to-end

## Suggested Solution
Build the minimum viable foundation: project scaffolding, core library, MCP server with basic tools, and a minimal CLI. No review pipeline, promotion, or advanced features — those are v0.2+.
