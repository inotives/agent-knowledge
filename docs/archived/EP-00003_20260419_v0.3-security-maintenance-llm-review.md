# EP-00003 — v0.3 Security, Maintenance & LLM Review

## Problem / Pain Points
- No protection against agents accidentally logging API keys, tokens, or credentials in turns or pages
- No maintenance tools for system health — no way to check for orphaned pages, stale content, or purge old sessions
- MCP prompts not implemented — agents have no playbook for session bootstrap/wrapup workflows
- `akw review` (LLM-powered daily review) not yet built — the automated bridge from session drafts to knowledge drafts
- CLI missing `akw purge` and `akw reindex` commands

## Suggested Solution
Add security safeguards, operational maintenance tools, MCP prompts for agent guidance, and the LLM-powered CLI review command. This makes the system production-ready.

## Implementation Phases

### Phase 1 — Secret Scanning
- [x] `src/agent_knowledge/core/sanitizer.py` — 10 secret patterns (AWS, API keys, bearer tokens, private keys, connection strings, GitHub/Anthropic/OpenAI tokens)
- [x] Integrated into `session_log`, `memory_create`, `memory_update` — auto-redacts secrets
- [x] 13 tests for pattern detection, redaction, and false positive avoidance

### Phase 2 — Maintenance Tools
- [x] `maintain_get_stats` MCP tool — page counts per tier, stale pages, session stats (total/reviewed/pending/orphaned)
- [x] `maintain_purge` MCP tool — deletes reviewed sessions + turns + draft files older than retention period
- [x] `akw purge [--older-than 365]` CLI command
- [x] `akw reindex` CLI command

### Phase 3 — MCP Prompts
- [x] `session_bootstrap` prompt — step-by-step guide for session start, pending review, context loading
- [x] `session_wrapup` prompt — guide for summarizing turns, writing session draft, ending session
- [x] Registered via FastMCP `@mcp.prompt()`

### Phase 4 — LLM-Powered CLI Review
- [x] Using `anthropic` SDK as runtime dependency (matches default config)
- [x] `akw review` — fetches pending sessions, calls Anthropic LLM, generates knowledge drafts + review report, marks sessions reviewed, cleans up session drafts
- [x] Secret redaction applied to LLM output before writing drafts
